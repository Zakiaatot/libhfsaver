# libhfsaver

A library for saving http-flv streams to disk.

## Prerequisites

1. Install needed packages:
   _Ubuntu_:

   ```bash
   sudo apt install cmake pkg-config ffmpeg libavcodec-dev libavformat-dev libavfilter-dev libavdevice-dev
   ```

2. Build:

   ```bash
   mkdir build && cd build
   cmake ..
   make
   ```
